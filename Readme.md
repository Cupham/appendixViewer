# appendixViewer: ECHONET Lite Appendix Viewer

2019.07.12    

## Abstract
appendixViewerは「APPENDIX ECHONET機器オブジェクト詳細規定」と同等の内容をブラウザーで表示するアプリケーションである

## Requirements
Web Browser  

## Installation
ファイルをダウンロードする

## Launch
htmlフォルダ内のindex.htmlをChromeなどのWeb Browserで開く
